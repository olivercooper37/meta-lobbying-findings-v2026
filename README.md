## Overview

Meta Lobbying and Other Findings is a Windows-based investigation tool built to organize lobbying activity, campaign finance data, and related public records into a clearer research workflow. It gathers open-source material, government filings, and archived references so users can move from scattered documents to a more traceable evidence trail.

The app is aimed at analysts, journalists, researchers, and other users who need to assemble a timeline around influence and policy activity. Keyword search, source links, and offline access make it easier to compare records, verify findings, and keep notes tied to referenced material.
